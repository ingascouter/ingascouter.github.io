    <p>Scoutavgiften går även till att utveckla Finlands scouter samt att ekonomiskt stötta alla finländska

    <h2>Rabatter på friluftsbutiker</h2>
    <p>Som medlem i scouterna får du <a href="https://scandinavianoutdoor.fi/partiolainen/">10% rabatt</a> på
        utvalda produkter i butikerna nedan. För att använda rabatten i
        butikerna
        kan du visa upp ditt medlemskort genom att skriva ut det eller bara öppna det via kuksa på din telefon.
        Om
        du
        vill
        använda rabatten på nätet behöver du registrera dig i webbutiken med ditt medlemsnummer som också hittas
        i
        kuksa.</p>
    <h3>Scandinavian Outdoor</h3>
    <p>Scandinavian Outdoor ger <a href="https://scandinavianoutdoor.fi/partiolainen/">10% rabatt</a> på allt
        förutom scoutprodukter, elektronik, presentkort, klubbpriser,
        produktpaket
        och kan inte kombineras med andra rabattkoder. Rabatten gäller både produkter till normalpris och
        kampanjpris.</p>
    <p>Finlands scouter samarbetar också med Scandinavian Outdoor och de ansvarar för försäljning och
        distribution
        av
        officiella scoutprodukter, såsom märken och riggar (se <a
            href="https://scandinavianoutdoor.fi/partio/">partio</a> på Scandinavian Outdoor).</p>
    <p><strong><i>Tips: studerande med studiekort får också 10% rabatt hos Scandinavian, men denna kan användas
                för
                alla
                produkter förutom de med klubbpriser.</i></strong></p>

    </p>
    <h3>Partioaitta</h3>
    <p>Partioaitta erbjuder också <a href="https://www.partioaitta.fi/yritys/suomen-partiolaiset/">10%
            rabatt</a>.
        Rabatten gäller inte nedsatta produkter, 365-klubberbjudanden, Second
        Hand-produkter och serviceprodukter.</p>
    <h2>Finlands vandrarhemsförbund</h2>
    <p>Som medlem i scouterna kan du <strong>kostnadsfritt</strong> beställa ett inhemskt vandrarhemskort som
        bland annat ger dig
        <strong>10% rabatt</strong> på
        boende på vandrarhem i Finland.
    </p>
    <p>Du kan även skaffa ett <a
            href="https://www.hostellit.fi/tuote/henkilokohtainen-kansainvalinen-hostellikortti-jarjestojasenelle/">internationellt
            vandrarhemskort till nedsatt pris</a>.</p>
    <p>Läs
        mer på
        <a href="https://hostellit.fi">hostellit.fi</a>, eller gå direkt till <a
            href="https://hostellijarjesto-lv.creamailer.fi/survey/answer/msphnh3g9xv5j">formuläret för gratis
            vandrarhemskort</a>.
    </p>
