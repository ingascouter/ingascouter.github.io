title: Hållbarhet
description: "Ingå Scouter har certifierats som en hållbar scoutkår av Finlands Svenska Scouter, vilket innebär att vi uppfyller vissa kriterier för hållbarhet. Nedan följer en lista med saker som vi fokuserar på inom vår kår."
---

<h2>Fostran</h2>
<ul>
    <li>Spara energi genom att släcka lampor och dylikt när de inte används</li>
    <li>Alla tar väl hand om utrustningen och man lär även ut hur man kan underhålla utrustning så att den
        ska vara så länge som
        möjligt</li>
    <li>Uppmuntra alla att vara med och påverka för en hållbar utveckling</li>
</ul>
<h2>Tillgänglighet</h2>
<ul>
    <li>Informera om möjligheten att ansöka om befrielse från medlemsavgiften</li>
    <li>Sprid information om kårens trygghetspersoner Jenny och Malin (2025)</li>
    <li>Upplys om kårens tillgängliga låneutrustning:</li>
    <ul>
        <li>Två mindre rinkkor och en sovsäck (januari 2025)</li>
    </ul>
</ul>
<h2>Måltider</h2>
<ul>
    <li>Använd det som redan finns i kårens skafferi innan något nytt köps in och se till att inget blir
        gammalt
        genom att först använda det vars sista förbrukningsdag infaller tidigast</li>
    <li>Utnyttja receptbanken och lägg gärna till egna recept på <a
            href="scout.fi/for-karen/karadministration/hallbarhet-i-karen/receptbanken">scout.fi/for-karen/karadministration/hallbarhet-i-karen/receptbanken</a>
    </li>
    <li>Erbjud möjligheten att ta med eventuellt överbliven mat hem efter evenemang för att undvika slöseri
    </li>
</ul>
<h2>Resor</h2>
<ul>
    <li>Hitta tillsammans i åldersgrupperna bra utfärdsmål i närheten som man kan besöka till fots eller med
        cykel</li>
    <li>Samåk alltid när det är möjligt, även till scoutmöten</li>
    <li>Prioritera hållbara färdmedel, som att gå eller cykla. För längre resor, försök använda buss eller
        tåg</li>
</ul>
<h2>Återvinning</h2>
<ul>
    <li>Se till att alla sorterar, och vet hur de gör det, i kårlokalen samt på evenemang</li>
    <li>Om kärlen är fulla kan de föras till sopstationen nere i småbåtshamnen</li>
    <li>Om platsen saknar sopsortering, sortera dem och för dem till närmaste återvinningsstation</li>
</ul>
